# 2D三角洲行动 (Delta Action 2D)
> 一款基于 HTML5 Canvas 开发的 2D 俯视角军事撤离射击游戏，融合物资搜刮、装备养成、BOSS 挑战、辐射区探索与局外经济成长体系，单文件即开即玩，无需任何依赖。


![License](https://img.shields.io/badge/license-MIT-blue?style=flat-square)



## 🎮 游戏简介
游戏以「战术搜刮 + 撤离生存」为核心玩法，玩家将从大厅携带装备进入超大地图，搜刮不同稀有度的物资箱，与 AI 士兵战斗，探索高风险高回报的辐射区，最终通过撤离点将物资带出对局，存入仓库并在黑市交易，实现局外成长。完整还原了军事撤离类游戏的核心体验，同时以轻量化 2D 俯视角呈现，打开浏览器即可游玩。

## 🖼️ 游戏预览
- 超大 20000×20000 无缝主地图 + 独立辐射区副地图
- 完整背包装备界面、物资搜刮弹窗、黑市商店系统
- 实时战斗特效、伤害数字、辐射状态视觉反馈
- 大厅 / 仓库 / 对局多场景无缝切换

## 🎯 核心特性
### 🔫 硬核战斗系统
- 15+ 种枪械全覆盖：手枪、冲锋枪、步枪、狙击枪、霰弹枪、轻机枪
- 真实枪械手感：后坐力、弹道散布、换弹读条、射速差异
- 分层护甲机制：5 级头盔 + 5 级防弹衣，独立耐久与伤害减免
- 丰富战斗反馈：击中粒子、伤害数字、低血量泛红、击杀提示

### 🎒 完整装备背包体系
- 5 种容量背包，支持武器、护甲、药品、道具分类管理
- 双武器槽位切换，支持快速切枪、丢弃物品、装备穿戴
- 7 种可使用药品，带读条动画与持续恢复效果
- 特殊道具：能量球（移速大幅提升）、辐射板、铀电池等

### 🗺️ 大地图自由探索
- 20000×20000 超大主地图，300 个障碍物构建复杂地形
- 200 个物资箱分布全图，6 种稀有度对应不同品质奖励
- 80+ AI 士兵全域巡逻，具备警戒、追击、射击、索敌完整行为逻辑
- 中心竞技场强力 BOSS，掉落传说级顶级装备

### ☢️ 辐射区特色玩法
- 主地图四角设置辐射区入口，进入独立 10000×10000 副地图
- 5 级辐射递增机制，从轻微影响到致命持续掉血
- 专属道具：辐射板阻止辐射等级上升，铀电池启动清洁室清零辐射
- 辐射区专属敌人与加密物资箱，死亡不掉落专属物资

### 🏪 局外经济与成长
- 钞票通用货币体系，物资出售、黑市购买全流通
- 个人仓库系统，支持对局物资存入、开局装备带入
- 黑市商店随机刷新商品，涵盖武器、护甲、药品、特殊道具
- 本地存档持久化，数据保存在浏览器本地，刷新不丢失

### 🚁 撤离核心玩法
- 地图固定撤离点，交互读条后成功撤离
- 成功撤离：所有携带物资完整存入仓库
- 对局死亡：掉落全部携带物资，仅加密箱物品保留

## 🧩 核心系统详解
### 1. 装备稀有度体系
共 5 个稀有度等级，品质越高属性越强、掉落概率越低：
- ⚪ **普通 (Common)**：基础装备与物资，常见于普通物资箱
- 🟢 **优秀 (Uncommon)**：小幅属性提升，普通箱概率产出
- 🔵 **稀有 (Rare)**：中期主流装备，高级物资箱稳定产出
- 🟣 **史诗 (Epic)**：高阶装备，辐射区与 BOSS 掉落为主
- 🟡 **传说 (Legendary)**：顶级属性，仅 BOSS 与辐射区加密箱产出

### 2. 辐射区机制
| 辐射等级 | 效果描述 | 视觉表现 |
|----------|----------|----------|
| Lv.0 | 无负面影响 | 轻微绿色滤镜 |
| Lv.1 | 轻微辐射，缓慢叠加等级 | 黄色屏幕边框 |
| Lv.2 | 中度辐射，移动速度小幅降低 | 橙色屏幕边框 |
| Lv.3 | 重度辐射，持续少量掉血 | 红色屏幕边框 |
| Lv.4 | 致命辐射，高额持续掉血 | 深红色全屏警示 |

> 携带**辐射板**可冻结辐射等级不再上升；前往辐射区中心清洁室，消耗**铀电池**可立即清零当前辐射等级。

### 3. 经济与存档规则
- **钞票获取**：出售仓库物资、对局内拾取变卖物
- **带入机制**：每局最多可从仓库选择 5 件物品带入对局
- **掉落规则**：对局死亡后，除加密箱内物品外全部掉落
- **存档方式**：基于浏览器 LocalStorage 本地存储，清除浏览器数据会丢失存档

## 🚀 快速开始
### 方式一：直接运行（推荐）
无需任何环境与依赖，下载项目后**双击 `index.html`**，使用浏览器打开即可直接开始游戏。

### 方式二：本地静态服务
如需局域网访问或解决部分浏览器本地文件限制，可启动简易静态服务：
```bash
# 方式1：Python 内置服务
python -m http.server 8080

# 方式2：Node.js http-server
npx http-server -p 8080
```
启动后在浏览器访问 `http://localhost:8080` 即可。

### 方式三：线上部署
支持一键部署到任意静态托管平台：
- GitHub Pages：将 `index.html` 上传至仓库，开启 Pages 服务
- Vercel / Netlify：直接拖拽文件即可完成部署
- 任意云服务器：放入 Nginx/Apache 静态目录即可访问

> 浏览器兼容性：支持 Chrome、Edge、Firefox、Safari 等主流现代浏览器，建议使用 Chromium 内核浏览器获得最佳体验。

## ⚙️ 自定义配置
游戏核心参数集中在代码顶部 `CONFIG` 对象中，可直接修改调整游戏体验：
```javascript
const CONFIG = {
    // 地图尺寸
    MAP_WIDTH: 20000,           // 主地图宽度
    MAP_HEIGHT: 20000,          // 主地图高度
    RADIATION_MAP_WIDTH: 10000, // 辐射区宽度
    RADIATION_MAP_HEIGHT: 10000,// 辐射区高度

    // 地图元素数量
    OBSTACLE_COUNT: 300,        // 障碍物数量
    LOOTBOX_COUNT: 200,         // 物资箱数量
    ENEMY_COUNT: 80,            // 普通敌人数量

    // 玩法参数
    SAFEBOX_CAPACITY: 10,       // 加密箱容量
    BRING_IN_LIMIT: 5,          // 开局带入物品上限
    RADIATION_DAMAGE: 2,        // 致命辐射每秒伤害
}
```

## 🛠️ 技术栈与实现
| 层级 | 技术方案 | 说明 |
|------|----------|------|
| 渲染层 | HTML5 Canvas 2D | 逐帧渲染游戏画面，相机跟随玩家视口裁剪 |
| 逻辑层 | 原生 JavaScript (ES6+) | 实现游戏状态机、AI 行为、碰撞检测、数值计算 |
| UI 层 | Tailwind CSS + DOM | 叠加式界面渲染，背包/商店/大厅基于 DOM 实现 |
| 数据层 | LocalStorage | 本地持久化存档，无需后端服务 |
| 性能优化 | 视口剔除 + 对象池 | 仅渲染屏幕范围内元素，复用敌人/粒子对象 |

## ❓ 常见问题 FAQ
**Q：游戏运行卡顿怎么办？**
A：可适当减小地图尺寸与敌人/障碍物数量；建议使用桌面端浏览器，移动端性能会有明显下降。

**Q：存档数据会丢失吗？**
A：数据保存在浏览器本地存储中，清除浏览器缓存、无痕模式下运行会导致存档丢失，重要数据请谨慎清理。

**Q：支持多人联机吗？**
A：当前版本为单机离线版本，暂不支持联机功能。

**Q：可以修改/二次创作这个游戏吗？**
A：在 MIT 协议范围内，你可以自由修改、分发、二次创作，无需授权。
